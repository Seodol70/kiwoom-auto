## 🎯 매수 신호 타입

### 1️⃣ BREAKOUT (단순 돌파 신호)

**상태:** 🔵 필터링용 (단독 매수 불가)

**조건:**

```markdown
✓ 전일종가 대비 1.5% 이상 상승 (2026-04-15 완화: 3%→1.5%, 진입 기회 확대)
✓ 당일 거래량 있음 (거래량 체크)
```

**목적:** 기초 여과 (BREAKOUT 통과 → JDM 추가 평가)  
**핵심:** 호가 몇 개 먹은 수준의 가짜 신호 배제, 의미 있는 움직임만 포착

**예시:**

```markdown
전일종가: 10,000원
현재가: 10,150원 이상 필요 (기존 10,300 → 하향)
거래량: > 0 필요
```

### 2️⃣ JDM_ENTRY (실제 매수 신호) ✅

**상태:** 🟢 매수 신호 (5가지 조건 모두 충족 필수)

#### 📋 조건 상세

| # | 조건 | 기준 | 설명 |
| :--- | :--- | :--- | :--- |
| 0 | **수급 절대치** | 거래대금 **상위 100위** 이내 | 2026-04-21 완화: 금액 제한 철폐, 상위 100위 이내 |
| 1 | **VWAP 생존 게이트** | **현재가 > VWAP * 0.995** | 2026-05-05 완화: 학습을 위해 -0.5% 버퍼 허용 |
| 2 | **MA 정배열 및 이격** | MA7 > MA15 교차 + **이격 ≥ 0.20%** | 2026-04-30 강화: 0.15% → 0.20% (Whipsaw 방지) |
| 3 | **거래량 급증** | 직전 10분 평균 × **1.2배 이상** | 2026-04-21 완화: 1.5배→1.2배 (태동기 포착) |
| 4 | **체결강도** | **110% 이상** | 2026-05-05 완화: 120% → 110% (공격적 진입) |
| 5 | **AI 스마트 판단** | **AI 스코어 > 0.3** | 2026-05-05 완화: 0.4 → 0.3 (학습 데이터 확보 모드) |
| 6 | **RSI** | **40~70% 범위** | 2026-05-05 하향: 45% → 40% (추세 Lv3 시 80% 완화) |

> **💡 스코어링 보너스 및 AI 가점**
>
> - **VWAP**: 현재가가 VWAP 위에 있어야만 AI 판단이 시작됨 (안전 펜스)
> - **MTF (Multi-Timeframe)**: 15분/60분봉 추세가 정배열일 경우 AI 가점 부여
> - **호가 잔량**: 매도잔량이 매수잔량보다 많을 때(물량 흡수 중) 돌파 확률 가점
> - **캔들 패턴**: 꼬리가 짧고 몸통이 긴 양봉 출현 시 가점

#### ⚙️ 기술 파라미터

## 💰 포지션 관리

### 손절 (Stop Loss)

```markdown
기준: -1.2% (일반 손절 - 3초 관찰) / -2.0% (하드 스탑 - 즉시 매도)
실행: OrderManager 실시간 모니터링
블랙리스트: 손절 발생 시 당일 재매수 무조건 차단
```

### 익절 (Take Profit)

```markdown
기준: +3.0% (기본 목표)
동적 트레일링: +1.5% 수익 도달 시 추격 시작 (고점 대비 0.8%~2.0% 하락 시 청산)
분할 익절: +2.0% 수익 도달 시 물량의 30% 우선 매도
```
